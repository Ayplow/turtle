Create Animated Drawings Quickly and Easily in Rust

Get started immediately and produce your first drawing within minutes. This crate is an excellent tool for learning and teaching the Rust programming language. Anyone of any age or skill level can learn how to create art with code!

Start Learning Now!

The Basic Concept

You control a turtle with a pen tied to its tail. When you tell it to move somewhere, it draws the path that it goes through to get there.

Change the color of the pen, fill shapes, and even use randomness to direct the turtle. There are plenty of ways to draw all kinds of amazing pictures!

Fun, and Easy to Learn Code

This crate is designed to be incredibly easy to learn. Each drawing command is designed to be simple and straightforward.

{% highlight rust linenos %} // This program draws a triangle let mut turtle = Turtle::new(); // Turn right 30 degrees turtle.right(30.0); for _ in 0..3 { // Walk forward 150 steps turtle.forward(150.0); turtle.right(120.0); } {% endhighlight %}
